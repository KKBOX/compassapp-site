Compass is a stylesheet authoring framework that makes your stylesheets and markup easier to build and maintain. With compass, you write your stylesheets in Sass instead of plain CSS.

Compass.app helps designers compile stylesheets easily without resorting to command line interface. It is written in Java and works in Windows/Linux/Mac OS X.

Features

  • Works in Windows/Linux/Mac
  • LiveReload support
  • Built-in Web server
  • Compass extensions support
